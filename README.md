# How started LOCALLY 🚀 ?

1. clone this repository : `git clone https://github.com/emmadal/codelntest`
2. cd `codelntest`
3. ```yarn install``` or ```npm install```
4. ```npm start```
5. Open your web browser at `http://localhost:3000` and enjoy 💻

## Online Link 🌍

open your browser at `https://codeln.netlify.com/`
